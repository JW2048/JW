02.面试流程 & 准备

发表于 2018-03-08 | 更新于 2018-07-30 | 分类于 求职 ， 02.面试

普通职位

// 面试流程（美团，其它公司类似）
面试一般为 4 轮，具体如下：
一轮（普通员工）
二轮（Leader）
三轮（总监）
四轮（HR）：薪资（有可能电话谈）

以上为一般情况，有可能不一样

// 面试准备
• 熟悉面试职位的职位要求
• 专业知识
可以去网站搜索面经
• 简历准备

  • 项目流程细节一定要非常熟悉
  • 想好怎么描述项目优化点

程序员

// 面试流程（美团，其它公司类似）
面试一般为 4 轮，具体如下：
一轮（普通）：基础知识 + 算法 + 项目
二轮（Leader）：算法 + 项目 + 项目优化
三轮（总监）：规划
四轮（HR）：薪资（有可能电话谈）

以上为一般情况，有可能不一样

// 面试准备
• 熟悉面试职位的职位要求
• 基础知识和算法
一般面试官也不会重复造轮子，所以复习下面的东西基本没问题。

• 专业知识
可以去网站搜索面经
• 简历准备

  • 技术点实现原理一定要复习
  • 项目流程细节一定要非常熟悉
  • 想好怎么描述项目优化点
