01.大纲

发表于 2018-03-08 | 更新于 2018-07-30 | 分类于 求职 ， 00.大纲

求职分为 3 部分：简历 + 面试 + 薪资

// 简历
简历直接决定你能否获取面试资格。很多面试者觉得自己能力很强，却收不到面试邀请，大部分原因就是简历写的太糟，你的简历不能脱颖而出，能力再强也无展示机会。

// 面试
面试不止需要个人能力，也有技巧，面试前的准备功课尤为重要 ，准备好了则能兵来将挡水来土掩，顺利通过。

// 薪资
面试通过，薪资的谈判也是门学问。钱要少了不甘心，钱要多了，用人单位也许会因为预算放弃你，那么，怎样为自己争取最大化利益呢？

步步为营，做好每一细节，你才有机会获得成功！
