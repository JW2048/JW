简历筛选规则（社招_研发）

Posted on 2018-03-08 | In 简历 , 简历筛选

如何博得 HR 和面试官的眼球，让简历脱颖而出，一般需要注意以下几点，写简历的时候也尽量往以下方向靠拢，有的没有的也可以想办法临时造一下，比如博客。

简历筛选原则

满足以下 2 项，或 1 项中较优秀的
• 学校（211、985毕业）
• 公司（一线互联网公司）
• 项目经验（B 端业务系统、ERP、CRM、SaaS）
• 学习意识（有 Blog 略深的原创、有代码风格意识）
• 特殊技术亮点：GitHub 有较多好评项目，参与开源项目，竞赛获奖等
• 推荐人特别好评

简历加分

学习

• 经常上 Stack Overflow 提问和回答
• 有读过两个以上开源基础组件源码
• 有贡献开源项目代码

个人产出

• 有原创博客
• 有写 GitHub 地址
• 有不错的个人作品
• 出于兴趣而不是工作需要写过项目或工具
• 在学校和前公司获奖较多

经验

• 有设计和主持项目经验
• 有带团队经验
• 有创业技术合伙人经验
